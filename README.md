## git コマンド
### clone
gitコマンドでcloneを使用することで、リポジトリをローカルのディレクトリにクローンしてくることができる。

#### 基本的な操作
```
// コマンド
$ git clone <git web URL>

// 例
$ git clone https://github.com/amanhirohisa/JavaVariableExtractor.git
```

#### リポジトリのリンク取得方法
1. clone対象となるリポジトリ画面から、「<> Code」を開く。<br>
2. 「HTTPS」のweb URLを取得。<br>
